#jmeter-plugins
====

包含功能如下:
```
1. Redis插件, 需要依赖: jedis.jar
2. MD5函数插件
3. HTTP接口自动签名前置处理器（接口参数按业务组装并MD5加密， 自动设置到消息头中）
